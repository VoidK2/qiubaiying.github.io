### 最大公约数、最小公倍数模版
第一弹
```
int gcd(int a,int b)  
{  
	if(a%b==0)  
		return b;  
	else  
		return gcd(b,a%b);  
}  

int lcm(int a,int b)  
{  
	return (a*b)/gcd(a,b);  
}
```

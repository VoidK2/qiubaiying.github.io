### **如何将一个顺序表中元素逆置**
本来还想用笨方法的，奈何是学算法的，还是要更高的效率
```c++
#define maxlen 1000
typedef struct{
	Datatype data[maxlen];
	int last;
}SequenList;
SequenList x,*L;
void SquenListInvert(SequenList *L){
	int i,temp;
	int n=L->last;
	for(i=0;i<n/2;i++){
		temp=L->data[i];
		L->data[i]=L->data[n-i-1];
		L->data[n-i-1]=temp;
	}
}
```
